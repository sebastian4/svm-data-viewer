- Type in a new task and press Enter or click on 'Add New.' Duplicate tasks are not added (the original gets highlighted in orange in case you're skeptical).

    - Double click on an existing task to edit it. Clear all text from a task when editing it to remove the task altogether. Edits that are duplicates don't update.

    - 'Clear list' removes all tasks forever. And ever.

    Made in jQuery. Everything is stored locally within your browser.
